# Studio 14 Frontend Assessment - Moyowa Etchie

## Overview

Live link: https://moyowa-studio14-assessment.vercel.app/

This project is a responsive single-page website for EAB , built as a solution for the Mid-Level Frontend Developer practical assessment. The implementation follows the Figma design specifications and demonstrates modern React development practices.

## 🛠 Tech Stack

- **React 19** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool and development server
- **Tailwind CSS** - Utility-first CSS framework
- **Vercel** - Deployment platform

## Features

- **Mobile-first approach** with breakpoints at md (768px) and lg (1024px) and "rem" values
- **Hover effects** on buttons and links
- **Mobile menu** hamburger icon on mobile
- **Image hover zoom** effect on hover
- **Smooth scrolling** for "Back to Top" functionality
- **Interactive carousel** in impact section (auto-play with manual controls)

### Styling & Design

- **Custom color palette** defined in Tailwind config
- **Multiple font families** (Source Serif, Source Sans Pro, Poppins)
- **Consistent spacing** using Tailwind's utility classes
- **Micro-interactions** with transition effects

## 🔧 Development Setup

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Moyowaaaa/studio14
cd studio14

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
